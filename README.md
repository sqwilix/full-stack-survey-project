Survey Project — это полнофункциональное Fullstack MERN-приложение (MongoDB, Express, React, Node.js) для создания, прохождения и управления онлайн-опросами.
Проект включает в себя админ-панель, защищённые маршруты, JWT-аутентификацию, обновление токенов, загрузку изображений, и полностью реализован с использованием React Router DOM на фронтенде.
🚀 Основные возможности

👤 Регистрация и авторизация пользователей (JWT + Refresh Token)

🔐 Защита маршрутов (только авторизованные пользователи могут видеть профиль)

🧠 Админ-панель:

Создание, редактирование и удаление опросов

Добавление вопросов и вариантов ответов

Загрузка изображений для опросов

🗳️ Публичная страница для прохождения опросов

📊 Просмотр результатов ответов пользователей

💾 Работа с MongoDB для хранения пользователей и опросов
